Intro

I am currently pursuing my undergraduate studies in Computer Science from Vivekanada Education Society's Institute of Technology, University of Mumbai. I have previosly worked on various technologies like Web Development, Blockchain, Image Processing using openCV and App development using Flutter. I love taking part in hackathons and programming contests. In the previous year's HacktoberFest hosted by digitalOcean I served as a mentor to a github repository. Open-source-projects really helped me to develop my skills, & I willingly contribute to it with pleasure.

As a CS student, learning new technologies and software is always top in my priority list. I recently got obsessed with Cybersecurity after watching confrences like DEF CON and Black Hat Breifings. Other than this coding stuff, you will mostly find me playing video games or vibing with my friends on netflix. Ocassionally I like to play guitar in the hope of playing one of the slash's guitar solos one day.

Learned to use various modules offered by the npm ecosystem and made a few ajax, jquery based scenarios along with some static pages using HTML/CSS/JS.

2. Project demonstrating the use of Sockets in a chat app
Project Demo

This project is a simple chat application. This project was made using nodejs on the server-side and HTML/CSS/JS for the frontend. Whenever a user joins or exits a chat room the server emits a message to that room. People can set their usernames and chat with other people in real-time.

3. Food Ordering Site

Project Report || Project Code

The project is made with HTML, CSS, JS, PHP, SQL. It is an online food ordering system as the name suggests. The backend is made with PHP while MySql is used for the database. The customer will be able to buy/add to cart multiple dishes from the menu and proceed to checkout. After checkout the order status is visible and a male of the order receipt is sent to the user's registered mail id.

4. Implementation of A* path finding algorithm

Project Code
